RHT v1.3.1

Welcome, to RHT's official webpage!

RHT is a tool for cracking router
passwords in Linux Kali and Backtrack.

To use RHT on a desktop computer you'll need a Wireless Network Adapter (WNA).

Laptops have a built-in wireless card that will do the trick.

Still the usage of WNA helps a lot for the succession of RHT's work.

Another thing you will need is Ruby,
the programmer's best friend.

Go to Ruby's website, download and install Ruby.

Kali comes with built-in Ruby 1.9.3,
but it's not fully reliable.

We suggest installing a new one.

You can get RHT by downloading the zip or by cloning the RHT repository.

Check the repository wiki
for more detailed information about
installation and starting of RHT.
